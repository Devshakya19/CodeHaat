## Logo Design

### Primary Logo — "The Hexagonal Node"

A modern, geometric logo that combines code and marketplace concepts. The hexagon represents connectivity and the tech world, while the inner design represents a code package being exchanged.

```
    ┌─────────────────────────────────────────┐
    │                                         │
    │   ╱╲                                    │
    │  ╱  ╲   ◆                              │
    │ ╱ ◇  ╲     CodeHaat                    │
    │ ╲    ╱                                 │
    │  ╲  ╱                                  │
    │   ╲╱                                   │
    │                                         │
    │   Hexagonal node with diamond center    │
    │   Represents: Code + Commerce exchange  │
    │                                         │
    └─────────────────────────────────────────┘
```

### Logo Specifications

| Element | Specification |
|---------|---------------|
| **Icon Size** | 32x32px (standard), 48x48px (large), 64x64px (hero) |
| **Icon Shape** | Hexagon (6-sided polygon) |
| **Icon Colors** | Primary: Slate 950 (`#0F172A`), Accent: Blue 500 (`#3B82F6`) |
| **Center Symbol** | Diamond/gem shape in Blue 500 (`#3B82F6`) |
| **Text Font** | Geist Sans (Bold, 20px) |
| **"Code" Color** | Slate 950 (`#0F172A`) |
| **"Haat" Color** | Slate 600 (`#475569`) |
| **Spacing** | 12px between icon and text |

### Logo Variants

| Variant | Icon | Text | Use Case |
|---------|------|------|----------|
| **Primary (Dark)** | Slate 950 hexagon + Blue diamond | Dark text | Light backgrounds |
| **Reversed (Light)** | White hexagon + Blue diamond | White text | Dark backgrounds (banners) |
| **Icon Only** | Slate 950 hexagon + Blue diamond | None | Favicons, app icons |
| **Monochrome** | Black hexagon + White diamond | Black | Print, single-color |

### Alternative Logo Concepts

#### Concept A: "Code Bracket"
Two angle brackets `< >` forming a diamond shape, representing code.

#### Concept B: "Marketplace Cube"
A 3D cube/package shape, representing the marketplace aspect.

#### Concept C: "Connected Nodes"
Three connected dots forming a triangle, representing the network of buyers, sellers, and code.

### Logo Usage Rules

1. **Minimum Size**: Icon must be at least 24x24px
2. **Clear Space**: Keep 1x icon width of clear space around the logo
3. **Background**: Use on contrasting backgrounds only
4. **Don't**: Stretch, rotate, or add effects to the logo

## Quick Start for Designers

1. Create a 1024x1024 canvas
2. Fill background with `#0F172A` (Slate 950)
3. Draw a hexagon shape (6-sided polygon) centered on canvas
4. Add a diamond/gem shape in the center with `#3B82F6` (Blue 500)
5. For full logo: Add "Code" (Slate 950) + "Haat" (Slate 600) text to the right
6. Export in SVG, PNG, and ICO formats
